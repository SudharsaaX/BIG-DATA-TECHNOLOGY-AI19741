# BIG DATA TECHNOLOGY (AI19741) – Laboratory Experiments

---

##  About  
This repository contains all **7 laboratory experiments** for the course *BIG DATA TECHNOLOGY (AI19741)*, as per the R2019 syllabus.  
It covers installation and usage of Hadoop ecosystem components (HDFS, MapReduce, Pig, Hive, Sqoop) and hands-on tasks to embed big data processing skills.

---

##  List of Experiments  
1. **Installation of Hadoop**  
2. **HDFS File-Management Tasks**  
   - Upload / Download a file in HDFS  
   - Copy a file from source → destination  
   - Copy file from/to local filesystem ↔ HDFS  
   - Move file from source → destination  
   - Remove a file/directory in HDFS  
3. **MapReduce – Word Count Program**  
4. **MapReduce – Weather Report (POC) Program**  
   - Analyze time-temperature statistics (max/min temperature)  
5. **Pig Latin Scripts**  
   - Sort, Group, Join, Project, Filter data  
6. **Hive** – Databases, Tables, Views, Functions & Indexes  
7. **Sqoop** – Export data from Hadoop using Sqoop & import data into Hive
